Output Examples

Depending on the image_format setting:

Link Format (Default)

All images become Markdown image syntax with clickable links:

![Beautiful landscape](https://example.com/images/landscape.jpg)
*Scenic mountain view*

![Company logo](images/logo.png)

Skip Format

All images are replaced with alt text placeholders:

*[Image: Beautiful landscape]*

*[Image: Company logo]*

Testing Instructions

To test image extraction:

  1. Convert with extract_images: false to see original HTML behavior
  2. Convert with extract_images: true, image_format: "link" to see linked images
  3. Convert with extract_images: true, image_format: "skip" to see alt text only
  4. Combine with extract_metadata: true to see full metadata extraction
