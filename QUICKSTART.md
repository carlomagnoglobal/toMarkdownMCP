# Quick Start

## Compile & Run in One Command

```bash
# Debug (faster to compile)
cargo run

# Release (optimized, slower to compile)
cargo run --release
```

## Common Tasks

| Task | Command |
|------|---------|
| Run tests | `cargo test` |
| Build only | `cargo build` |
| View help | `cargo run -- --help` |
| List tools | `cargo run -- tools` |
| Convert file | `cargo run -- convert FILE` |
| Search vault | `cargo run -- search QUERY` |

See [BUILD.md](BUILD.md) for detailed instructions.
